---
title: " Renovar P2P"
api: "PUT /p2p/extend/{id}"
description: "Renova um usuário P2P"
---

## Endpoint

<CodeGroup>

```bash Requisição
PUT /p2p/extend/{id}
```

</CodeGroup>

## Descrição

Renova um usuário P2P existente, estendendo sua validade.

## Parâmetros de URL

<ParamField path="id" type="integer" required>
  ID do usuário P2P
</ParamField>

## Parâmetros do Body

<ParamField body="extend_days" type="integer" required>
  Número de dias para estender
</ParamField>

<ParamField body="package_id" type="integer">
  Novo ID do pacote (opcional)
</ParamField>

<ParamField body="max_devices" type="integer">
  Novo limite de dispositivos (opcional)
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      ID do usuário P2P
    </ResponseField>
    <ResponseField name="username" type="string">
      Nome de usuário
    </ResponseField>
    <ResponseField name="previous_expires_at" type="string">
      Data de expiração anterior
    </ResponseField>
    <ResponseField name="new_expires_at" type="string">
      Nova data de expiração
    </ResponseField>
    <ResponseField name="days_extended" type="integer">
      Dias estendidos
    </ResponseField>
    <ResponseField name="max_devices" type="integer">
      Novo limite de dispositivos
    </ResponseField>
  </Expandable>
</ResponseField>
