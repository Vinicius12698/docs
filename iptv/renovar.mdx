---
title: " Renovar IPTV"
api: "PUT https://api-new.paineloffice.click/iptv/extend/{id}"
description: "Renova um usuário IPTV"
auth: "none"
---

## Endpoint

<CodeGroup>

```bash Requisição
PUT /iptv/extend/{id}
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Renova um usuário IPTV existente, estendendo sua validade.

## Parâmetros

<ParamField query="username" type="string" required>
  Nome de usuário para autenticação
</ParamField>

<ParamField query="password" type="string" required>
  Senha para autenticação
</ParamField>

<ParamField query="token" type="string" required>
  Token de autenticação
</ParamField>

<ParamField body="amount" type="number" required>
  Valor da renovação
</ParamField>

<ParamField body="month" type="number" required>
  Duração em meses
</ParamField>

<ParamField body="screen" type="integer" required>
  Só é possivel apenas 4 telas 
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      ID do usuário IPTV
    </ResponseField>
    <ResponseField name="username" type="string">
      Nome de usuário
    </ResponseField>
    <ResponseField name="previous_expires_at" type="string">
      Data de expiração anterior
    </ResponseField>
    <ResponseField name="new_expires_at" type="string">
      Nova data de expiração
    </ResponseField>
    <ResponseField name="days_extended" type="integer">
      Dias estendidos
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Requisição

<CodeGroup>

```json TryIt
{
```

```bash cURL
curl -X PUT "https://api-new.paineloffice.click/iptv/extend/1001" \
```
