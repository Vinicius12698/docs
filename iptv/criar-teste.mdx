## Endpoint

<CodeGroup>

```bash Requisição
POST /iptv
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Cria um novo teste IPTV gratuito no sistema.

## Parâmetros

<ParamField query="username" type="string" required>
  Nome de usuário para autenticação
</ParamField>

<ParamField query="password" type="string" required>
  Senha para autenticação
</ParamField>

<ParamField query="token" type="string" required>
  Token de autenticação
</ParamField>

<ParamField body="isTrial" type="boolean" required default="true">
  Sempre true
</ParamField>

<ParamField body="screen" type="integer" required default="1">
  Número máximo de conexões simultâneas (fixo em 1)
</ParamField>

<ParamField body="packageId" type="integer" required>
  ID do pacote IPTV
</ParamField>

<ParamField body="whatsapp" type="string">
  Número do WhatsApp (opcional)
</ParamField>

<ParamField body="notes" type="string">
  Observações adicionais
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      ID do teste IPTV criado
    </ResponseField>
    <ResponseField name="username" type="string">
      Nome de usuário IPTV
    </ResponseField>
    <ResponseField name="max_connections" type="integer">
      Número máximo de conexões
    </ResponseField>
    <ResponseField name="status" type="string">
      Status do teste
    </ResponseField>
    <ResponseField name="expires_at" type="string">
      Data de expiração do teste
    </ResponseField>
    <ResponseField name="connection_url" type="string">
      URL de conexão IPTV
    </ResponseField>
    <ResponseField name="m3u_url" type="string">
      URL do arquivo M3U
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Requisição

<CodeGroup>

```json TryIt
{
  "isTrial": true,
  "notes": "teste-auto",
  "packageId": 24,
  "screen": 1,
  "whatsapp": ""
}
```

```bash cURL
curl -X POST "https://api-new.paineloffice.click/iptv" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "isTrial": true,
    "notes": "teste-auto",
    "packageId": 24,
    "screen": 1,
    "whatsapp": ""
  }'
```

</CodeGroup>
