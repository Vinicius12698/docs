## Endpoint

### Recreate Usuário IPTV

<CodeGroup>

```bash Requisição
POST /iptv/recreate/:id
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Recreate um usuário IPTV existente, redefinindo suas configurações e credenciais.

## Parâmetros

### Path Parameters

<ParamField path="id" type="string" required>
  ID do usuário IPTV a ser recreate
</ParamField>

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token para autenticação
</ParamField>

### Body Parameters

<ParamField body="pass" type="string" required>
  Nova senha do usuário IPTV
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      ID do usuário IPTV recreate
    </ResponseField>
    <ResponseField name="username" type="string">
      Nome de usuário IPTV
    </ResponseField>
    <ResponseField name="password" type="string">
      Nova senha do usuário
    </ResponseField>
    <ResponseField name="status" type="string">
      Status do usuário
    </ResponseField>
    <ResponseField name="updatedAt" type="string">
      Data e hora do recreate
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Requisição

<CodeGroup>

```json TryIt
{
  "pass": "novaSenha123"
}
```

```bash cURL
curl -X POST "https://api-new.paineloffice.click/iptv/recreate/{ID}" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "pass": "novaSenha123"
  }'
```

</CodeGroup>
