## Endpoint

<CodeGroup>

```bash Requisição
POST /red-club/extend/{id}
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Estende a validade de um usuário RedClub existente.

## Parâmetros

<ParamField query="username" type="string" required>
  Nome de usuário para autenticação
</ParamField>

<ParamField query="password" type="string" required>
  Senha para autenticação
</ParamField>

<ParamField query="token" type="string" required>
  Token de autenticação
</ParamField>

<ParamField path="id" type="string" required>
  ID do usuário RedClub
</ParamField>

<ParamField query="credits" type="number" required>
  Créditos para extensão
</ParamField>

<ParamField query="screens" type="integer" required>
  Número de telas
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X POST "https://api-new.paineloffice.click/red-club/extend/{id}?credits=1&screens=1" \
  -H "Authorization: Bearer <token>"
```

</CodeGroup>
