## Endpoint

<CodeGroup>

```bash Requisição
POST /red-club
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Cria um novo usuário RedClub no sistema.

## Parâmetros

<ParamField query="username" type="string" required>
  Nome de usuário para autenticação
</ParamField>

<ParamField query="password" type="string" required>
  Senha para autenticação
</ParamField>

<ParamField query="token" type="string" required>
  Token de autenticação
</ParamField>

<ParamField body="email" type="string" required>
  E-mail do usuário
</ParamField>

<ParamField body="enabled" type="boolean" required>
  Se o usuário está habilitado
</ParamField>

<ParamField body="months" type="string" required>
  Duração em meses
</ParamField>

<ParamField body="notes" type="string">
  Observações
</ParamField>

<ParamField body="password" type="string" required>
  Senha do usuário
</ParamField>

<ParamField body="sale_value" type="number" required>
  Valor da venda
</ParamField>

<ParamField body="trial" type="boolean" required>
  Se é um teste gratuito
</ParamField>

<ParamField body="username" type="string" required>
  Nome de usuário RedClub
</ParamField>

<ParamField body="whatsapp" type="number" required>
  WhatsApp do usuário (apenas números)
</ParamField>
