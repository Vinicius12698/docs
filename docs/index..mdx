## 1. Autenticação

A autenticação é feita através do endpoint `POST /auth/token`. Para obter um token de acesso, basta enviar suas credenciais (usuário e senha) e você receberá um token válido.

Para mais detalhes, consulte a seção de [Autenticação](autenticacao.mdx).

## Códigos de Erro

A seguir estão os códigos de erro comuns retornados pela API:

- **400**: Requisição inválida.
- **401**: Não autorizado.
- **404**: Não encontrado.
- **500**: Erro interno do servidor.
