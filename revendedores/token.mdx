## Endpoint

<CodeGroup>

```bash Requisição
PUT /resale/token/fixed
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Fixa ou libera o token de uma revenda específica.

## Parâmetros

<ParamField header="Authorization" type="string" required>
  Bearer token para autenticação
</ParamField>

<ParamField body="id" type="integer" required>
  ID da revenda
</ParamField>

<ParamField body="success" type="boolean" required>
  Status de sucesso da operação
</ParamField>

<ParamField body="is_fixed" type="integer" required>
  Status do token (1 para fixar, 0 para liberar)
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="resale_id" type="integer">
      ID da revenda
    </ResponseField>
    <ResponseField name="token_fixed" type="boolean">
      Status do token (fixado/liberado)
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      Data da atualização
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Resposta

**Sucesso (200 OK):**
```json
{
  "success": true,
  "message": "Token fixado com sucesso",
  "data": {
    "resale_id": 15,
    "token_fixed": true,
    "updated_at": "2024-01-15T11:30:00Z"
  }
}
```
