## Endpoint

<CodeGroup>

```bash Requisição
PATCH /resale/ultra/{resaleId}/active
```

</CodeGroup>

## Descrição

Ativa o serviço Ultra para uma revenda específica.

## Parâmetros de URL

<ParamField path="resaleId" type="integer" required>
  ID da revenda
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="resale_id" type="integer">
      ID da revenda
    </ResponseField>
    <ResponseField name="ultra_active" type="boolean">
      Status do serviço Ultra
    </ResponseField>
    <ResponseField name="activated_at" type="string">
      Data de ativação
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Resposta

**Sucesso (200 OK):**
```json
{
  "success": true,
  "message": "Ultra ativado com sucesso",
  "data": {
    "resale_id": 15,
    "ultra_active": true,
    "activated_at": "2024-01-15T12:00:00Z"
  }
}
```
