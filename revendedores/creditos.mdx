---
title: " Gerenciar Créditos"
api: "PATCH /resale/add-credits/{id}"
description: "Adiciona ou remove créditos de uma revenda"
---

## Endpoint

<CodeGroup>

```bash Requisição
PATCH /resale/add-credits/{id}
```

</CodeGroup>

## Descrição

Adiciona ou remove créditos de uma revenda específica.

## Parâmetros de URL

<ParamField path="id" type="integer" required>
  ID da revenda
</ParamField>

## Parâmetros do Body

<ParamField body="credits" type="number" required>
  Quantidade de créditos (positivo para adicionar, negativo para remover)
</ParamField>

<ParamField body="reason" type="string">
  Motivo da alteração
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      ID da revenda
    </ResponseField>
    <ResponseField name="previous_credits" type="number">
      Créditos anteriores
    </ResponseField>
    <ResponseField name="new_credits" type="number">
      Novos créditos
    </ResponseField>
    <ResponseField name="credits_added" type="number">
      Créditos adicionados/removidos
    </ResponseField>
    <ResponseField name="reason" type="string">
      Motivo da alteração
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Resposta

**Sucesso (200 OK):**
```json
{
  "success": true,
  "message": "Créditos atualizados com sucesso",
  "data": {
    "id": 15,
    "previous_credits": 100,
    "new_credits": 150,
    "credits_added": 50,
    "reason": "Recarga manual"
  }
}
```
