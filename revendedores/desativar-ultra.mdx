## Endpoint

<CodeGroup>

```bash Requisição
PATCH /resale/ultra/{resaleId}/disable
```

</CodeGroup>

## Descrição

Desativa o serviço Ultra para uma revenda específica.

## Parâmetros de URL

<ParamField path="resaleId" type="integer" required>
  ID da revenda
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="resale_id" type="integer">
      ID da revenda
    </ResponseField>
    <ResponseField name="ultra_active" type="boolean">
      Status do serviço Ultra
    </ResponseField>
    <ResponseField name="disabled_at" type="string">
      Data de desativação
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Resposta

**Sucesso (200 OK):**
```json
{
  "success": true,
  "message": "Ultra desativado com sucesso",
  "data": {
    "resale_id": 15,
    "ultra_active": false,
    "disabled_at": "2024-01-15T12:30:00Z"
  }
}
```
