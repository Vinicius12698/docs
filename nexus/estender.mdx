## Endpoint

<CodeGroup>

```bash Requisição
POST /nexus/extend
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Estende a validade de um usuário Nexus existente.

## Parâmetros

<ParamField query="username" type="string" required>
  Nome de usuário para autenticação
</ParamField>

<ParamField query="password" type="string" required>
  Senha para autenticação
</ParamField>

<ParamField query="token" type="string" required>
  Token de autenticação
</ParamField>

## Parâmetros do Body

<ParamField body="credits" type="number" required>
  Créditos para extensão
</ParamField>

<ParamField body="screens" type="integer" required>
  Número de telas
</ParamField>

<ParamField body="username" type="string" required>
  Nome de usuário Nexus
</ParamField>

<ParamField body="password" type="string" required>
  Senha do usuário
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```json TryIt
{
  "credits": 0.5,
  "screens": 1,
  "username": "mirelateste",
  "password": "teste123"
}
```

```bash cURL
curl -X POST "https://api-new.paineloffice.click/nexus/extend" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "credits": 0.5,
    "screens": 1,
    "username": "mirelateste",
    "password": "teste123"
  }'
```

</CodeGroup>
