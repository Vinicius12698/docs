## Endpoint

<CodeGroup>

```bash Requisição
POST /nexus
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Cria um novo usuário Nexus no sistema. Esta é a mesma rota usada para criar testes Nexus, mas com `trial: false` para usuários pagos.

## Parâmetros

<ParamField header="Authorization" type="string" required>
  Bearer token para autenticação
</ParamField>

<ParamField body="username" type="string" required>
  Nome de usuário Nexus
</ParamField>

<ParamField body="password" type="string" required>
  Senha do usuário
</ParamField>

<ParamField body="whatsapp" type="number">
  WhatsApp do usuário (apenas números)
</ParamField>

<ParamField body="notes" type="string">
  Observações
</ParamField>

<ParamField body="email" type="string">
  E-mail do usuário
</ParamField>

<ParamField body="months" type="string" required>
  Duração em meses
</ParamField>

<ParamField body="sale_value" type="number">
  Valor da venda
</ParamField>

<ParamField body="screens" type="integer" required>
  aplicar descrição de máximo de até 2 telas
</ParamField>

<ParamField body="trial" type="boolean" required default="false">
  Para criação de usuário, mantenha false
</ParamField>
