## Endpoint

<CodeGroup>

```bash Requisição
PATCH /clientes/editar/{id}
```

</CodeGroup>

## Descrição

Este endpoint permite editar os dados de um cliente existente.

## Parâmetros de URL

<ParamField path="id" type="string" required>
  ID do cliente a ser editado
</ParamField>

## Parâmetros do Body

<ParamField body="nome" type="string">
  Nome do cliente
</ParamField>

<ParamField body="email" type="string">
  Email do cliente
</ParamField>

<ParamField body="telefone" type="string">
  Telefone do cliente
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      ID do cliente editado
    </ResponseField>
    <ResponseField name="nome" type="string">
      Nome atualizado do cliente
    </ResponseField>
    <ResponseField name="email" type="string">
      Email atualizado do cliente
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Resposta

```json
{
  "success": true,
  "data": {
    "id": "12345",
    "nome": "João Silva Santos",
    "email": "joao.santos@email.com"
  }
}
```
