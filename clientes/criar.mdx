## Endpoint

<CodeGroup>

```bash Requisição
POST /clientes/criar
```

</CodeGroup>

## Descrição

Este endpoint permite criar um novo cliente no sistema.

## Parâmetros

<ParamField body="nome" type="string" required>
  Nome do cliente
</ParamField>

<ParamField body="email" type="string" required>
  Email do cliente
</ParamField>

<ParamField body="telefone" type="string">
  Telefone do cliente
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      ID do cliente criado
    </ResponseField>
    <ResponseField name="nome" type="string">
      Nome do cliente
    </ResponseField>
    <ResponseField name="email" type="string">
      Email do cliente
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Resposta

```json
{
  "success": true,
  "data": {
    "id": "12345",
    "nome": "João Silva",
    "email": "joao@email.com"
  }
}
```
